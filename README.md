# TeleBotAI

TeleBotAI — An AI-powered Telegram bot framework designed for automation, NLP-driven interactions, and scalable backend integration.

- start - Initialize the bot and show welcome message
- status - Get overall system status of all servers
- check - Check health and metrics of a specific server
- help - Show available commands and usage
- gather - Start an interactive session to collect user information (name, email, phone)

---

- start - Lets create a new Instance at vast.ai
